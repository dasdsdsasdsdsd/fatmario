# Skyline Ops

Skyline Ops is an **original** 3D FPS training-range prototype built with HTML, CSS, and Three.js. It uses a fullscreen canvas with a simple join screen so you can enter a username and loadout before deploying.

## How to run

Open `index.html` in a browser or serve it locally:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls

- Move: WASD
- Look: Mouse
- Sprint: Shift
- Fire: Left click
- Reload: R
- Join: J
- Pause/Release mouse: Esc
