# Skyline Ops

Skyline Ops is an **original** 3D FPS training-range prototype built with HTML, CSS, and Three.js. It is not a clone of any copyrighted game; it focuses on a safe, custom experience with its own mechanics and visuals.

## How to run

Open `index.html` in a browser or serve it locally:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Controls

- Move: WASD
- Look: Mouse
- Sprint: Shift
- Fire: Left click
- Reload: R
- Pause/Release mouse: Esc
